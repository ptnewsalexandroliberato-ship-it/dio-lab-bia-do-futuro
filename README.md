# 💰 Edu: Mentor de Educação Financeira com IA Local

> **Status:** Finalizado (Challenge DIO)  
> **Stack:** Python, Streamlit, Ollama (Modelo gpt-oss).

## 🧠 O Conceito
O **Edu** é um agente inteligente de educação financeira que utiliza a técnica de **RAG (Retrieval-Augmented Generation)** para analisar dados locais de perfil e transações sem comprometer a privacidade do usuário. Ele não é um consultor de investimentos, mas um educador focado em literacia financeira.

## 🚀 Diferenciais Intelectuais (Novidades 2026)
- **Privacidade Local-First:** Todo o processamento de linguagem natural ocorre via Ollama. Seus dados financeiros nunca saem da sua máquina.
- **Análise Proativa de Gastos:** O Edu calcula automaticamente o custo de vida baseando-se no histórico de transações (`.csv`) e sugere metas educativas.
- **Protocolo Ético de Não-Recomendação:** Blindagem via System Prompt para garantir que o agente atue apenas como professor, nunca como corretor.

## 🛠️ Como Executar
1. Certifique-se de ter o **Ollama** instalado e rodando o modelo `gpt-oss`.
2. Instale as dependências: `pip install -r requirements.txt`
3. Inicie a interface: `streamlit run app.py`
